# RevProxy - Simple Reverse Proxy

## Feature

### Simple Usage

```bash
revproxy --listen=8000 --address="http://google.com"
```

### Header Authentication

```bash
revproxy --listen=8000 --address="http://google.com" --password="IIPZBLaSLeXqpefCPalPObPqiwPXmQMMpP+E4U1h1nA="
```
